Crypto-Feed APP

Full-stack MERN app using API's to access crypto data

Tech Used:

-MongoDb

-Express.js

-React.js

-Tailwind CSS

-Visual Studio Code

-JWT

-Heroku

Installation Instructions

-Fork and clone this repository to your local machine

-Install necessary packages

npm i

Run Frontend:

npm run frontend

Run Backend:

npm run backend
